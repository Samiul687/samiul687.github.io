Projects

Sudoku Solver

This was coded using backtracking algorithm. The programme finds empty spaces and then attempts to find a match before moving onto the next cell. If the solution is invalid, the program will return to previous cells and repeat the process.

Armstring Number Checker

This programme uses a generator expression to split a number. I chose to use this instead of list comprehension as I won't need to access the data afterwards by index. After splitting an integer into it's individual digits, it will find out whether or not this integer is an Armstrong number.

Pythagorean Triples Checker

The user inputs the lengths of all 3 sides of a triangle seperated by a comma. The programme will then split the user's input into the different sides and sort them in order of lowest to highest. Finally, it will apply Pythagoras' Theorem to check if they are Pythagorean Triples.
